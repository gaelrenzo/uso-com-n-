# Frontend Design

Anthropic's official skill for frontend design. Build beautiful, functional web interfaces.

## When to Use

- Designing or building web UIs
- Creating responsive layouts
- Implementing design systems
- Building interactive components

## Capabilities

- HTML/CSS/JS generation
- Responsive design patterns
- Component architecture
- Accessibility best practices
- Animation and transitions
- Modern CSS (Grid, Flexbox, Container Queries)

## Quick Start

```
# Create a landing page
/build a landing page for [product]

# Design a component
/create a responsive navigation component

# Build a dashboard
/design an analytics dashboard with charts
```

## Best Practices

- Mobile-first design
- Semantic HTML
- ARIA accessibility
- Performance optimization
- Cross-browser compatibility
