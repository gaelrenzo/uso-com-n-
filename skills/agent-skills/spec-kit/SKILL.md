# spec-kit

Spec-Driven Development toolkit by GitHub. Specifications become executable, generating working implementations.

## When to Use

- User wants to bootstrap a project with spec-driven methodology
- Creating specifications, plans, and task breakdowns
- Setting up AI coding agent integrations (Claude, Codex, Copilot, Cursor, Gemini)

## Overview

CLI that bootstraps projects with spec templates, plans, tasks, and integrations for 30+ AI coding agents. Workflow: constitution → specify → plan → tasks → implement.

## Usage

```bash
# Install
pip install spec-kit

# Initialize a project
spec-kit init

# Create a specification
spec-kit specify "feature name"

# Generate implementation plan
spec-kit plan

# Break into tasks
spec-kit tasks
```

## Workflow

1. **Constitution** - Define project principles and constraints
2. **Specify** - Write detailed specifications
3. **Plan** - Generate implementation plan
4. **Tasks** - Break plan into actionable tasks
5. **Implement** - Execute with AI agents
