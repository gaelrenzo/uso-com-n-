# last30days

AI agent-led search engine. Searches Reddit, X, YouTube, TikTok, HN, Polymarket, GitHub and 10+ platforms in parallel.

## When to Use

- User wants to research a topic across multiple platforms
- Finding recent discussions, trends, or opinions
- Competitive analysis or market research
- Slash command: `/last30days <topic>`

## Platforms Searched

- Reddit
- X (Twitter)
- YouTube
- TikTok
- Hacker News
- Polymarket
- GitHub
- And 10+ more

## Features

- Parallel multi-platform search
- Real engagement scoring (upvotes, likes, real money)
- Synthesized brief generation
- HTML export
- Competitor comparisons
- GitHub person-mode

## Usage

```
/last30days <topic>
```
