# Obsidian Skills

Agent skills for Obsidian vault management. 5 skills for markdown, databases, diagrams, CLI, and web extraction.

## Skills Included

### obsidian-markdown
Wikilinks, embeds, callouts, properties, and Obsidian-specific markdown syntax.

### obsidian-bases
Database views, filters, formulas, and property-based queries.

### json-canvas
Node/edge diagram creation for visual mappings.

### obsidian-cli
CLI interaction with Obsidian vaults — open, search, create notes.

### defuddle
Web page to clean markdown extraction for Obsidian.

## When to Use

- Managing Obsidian vaults with AI agents
- Creating notes with proper Obsidian syntax
- Building diagrams or database views
- Extracting web content into clean markdown
