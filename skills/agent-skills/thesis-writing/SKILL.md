# thesis-writing

AI agent skill for scoping, structuring, drafting, and reviewing empirical theses and dissertations.

## When to Use

- Writing academic papers, theses, or dissertations
- Need help with structure, outline, or drafting
- Review and feedback on academic writing
- Undergraduate through doctoral level

## Modes

| Mode | Purpose |
|------|---------|
| **define/scope** | Define research question and scope |
| **outline** | Create document structure |
| **draft** | Write content sections |
| **review** | Get feedback and improvements |
| **process/planning** | Manage writing workflow |

## Parameters

- **level**: undergraduate | masters | doctoral
- **language**: any natural language
- **format**: Markdown | Word | LaTeX

## Quick Start

```
# Start a new thesis
/help me scope a thesis on [topic]

# Create outline
/create an outline for my thesis

# Draft a section
/draft the literature review section

# Review my writing
/review this chapter for clarity
```
