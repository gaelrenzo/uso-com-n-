# FreeDomain Skill

Register and manage free domain names through DigitalPlat FreeDomain.

## When to Use

- User wants a free domain name (.dpdns.org, .us.kg, .qzz.io, .xx.kg, .qd.je)
- User asks about DNS hosting or domain registration
- User needs help configuring a free domain with Cloudflare or other DNS providers

## Overview

DigitalPlat FreeDomain provides free domain names to everyone. Over 500,000 domains registered.

### Available Extensions

- `.dpdns.org`
- `.us.kg`
- `.qzz.io`
- `.xx.kg`
- `.qd.je`

## Workflow

1. Register account at the dashboard
2. Choose a domain name and extension
3. Set DNS nameservers (e.g., Cloudflare, FreeDNS by Afraid.org, or Hostry)
4. Configure DNS records at your chosen provider
