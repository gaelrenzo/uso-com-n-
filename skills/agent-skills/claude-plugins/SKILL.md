# Claude Plugins Official

Anthropic's curated directory of official and third-party plugins for Claude Code.

## When to Use

- User wants to install or configure Claude Code plugins
- Looking for integrations (Telegram, Discord, iMessage)
- Managing plugin lifecycle (install, update, remove)

## Available Plugins

### Official
- **claude-md-management** - Manage CLAUDE.md files across projects
- **code-modernization** - Update legacy code patterns
- **security-guidance** - Security best practices
- **session-report** - Generate session summaries
- **skill-creator** - Create new skills

### Third-party
- Telegram integration
- Discord integration
- iMessage integration

## Installation

```bash
# Via Claude Code
/install-plugin <plugin-name>

# Via CLI
claude plugin install <plugin-name>
```
