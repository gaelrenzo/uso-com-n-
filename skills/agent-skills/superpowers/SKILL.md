# Superpowers

Complete software development methodology for coding agents. Composable skills enforce spec-driven workflow.

## When to Use

- Starting a new development project
- User wants structured, test-driven development
- Dispatching parallel subagents for tasks
- Code review and verification workflows

## Skills Included (14)

| Skill | Purpose |
|-------|---------|
| brainstorming | Explore ideas before coding |
| writing-specs | Create detailed specifications |
| writing-plans | Build implementation plans |
| executing-plans | Follow plans step-by-step |
| test-driven-development | Write tests first |
| systematic-debugging | Find root causes |
| dispatching-parallel-agents | Run tasks in parallel |
| subagent-driven-development | Delegate to subagents |
| verification-before-completion | Validate before marking done |
| receiving-code-review | Handle feedback |
| requesting-code-review | Ask for reviews |
| using-git-worktrees | Parallel branch work |
| finishing-a-development-branch | Clean merge workflow |
| writing-skills | Create new skills |

## Workflow

1. **Brainstorm** - Explore what to build
2. **Spec** - Define requirements
3. **Plan** - Break into tasks
4. **Implement** - TDD with subagents
5. **Verify** - Check before completion
6. **Review** - Code review cycle
7. **Finish** - Clean merge
