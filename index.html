    Proxy local para IA

    Free Claude Code - Setup y Uso

    1. Iniciar el Servidor Proxy

    Ejecuta el servidor proxy en segundo plano en el puerto indicado.

    # Para Windows (PowerShell):$env:Path = "C:\Users\renzo\.local\bin;$env:Path"fcc-server --port 8082# Para Linux / Android Termux (Ubuntu):fcc-server --port 8082

    2. Configurar en el Admin UI

    Abre el panel de administración local en tu navegador para configurar las API Keys de los proveedores (ej: Gemini, OpenRouter).

    URL de administracion:http://127.0.0.1:8082/admin# 1. Pega tu API Key en GEMINI_API_KEY# 2. Configura MODEL con gemini/models/gemini-2.5-flash# 3. Presiona Validate y luego Apply

    3. Ejecutar Claude Code

    Inicia el asistente Claude Code configurado para usar el proxy local de Inteligencia Artificial.

    # Para Windows (PowerShell):fcc-claude# Para Linux / Android Termux (Ubuntu):fcc-claude

    4. Integración en VS Code (settings.json)

    Configura la extensión de Claude Code en tu editor para redirigir sus peticiones al proxy.

    "claudeCode.environmentVariables": [  { "name": "ANTHROPIC_BASE_URL", "value": "http://localhost:8082" },  { "name": "ANTHROPIC_AUTH_TOKEN", "value": "freecc" },  { "name": "CLAUDE_CODE_ENABLE_GATEWAY_MODEL_DISCOVERY", "value": "1" },  { "name": "CLAUDE_CODE_AUTO_COMPACT_WINDOW", "value": "190000" }]
